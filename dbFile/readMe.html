Project Setup Instructions

  1. Download the ZIP file of the project.
  2. Extract the project into C:\xampp\htdocs.
  3. Go to http://localhost/phpmyadmin/ and import the database from (dbFile\db_project.sql).
  4. In your browser, go to http://localhost/DB_PROJECT/login.php to start working.
  5. If you have any questions or need changes, please suggest via WhatsApp.
